Tutorial Minecraft Original

Langkah-langkah Join Server:

  1. Setelah game terbuka, klik tombol "Multiplayer".
  2. Klik tombol "Add Server" di bagian bawah.
  3. Untuk "Server Name", isi bebas (Contoh: Server Keren).
  4. Untuk "Server Address", masukkan IP berikut:
    mc.stoy.biz.id
  5. Klik "Done", lalu cari servernya dan klik "Join Server".
